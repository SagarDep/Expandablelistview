# Expandablelistview
就是简单的一个ExpandableListView的使用，只有两级列表，然后再搭配一个时间选择器进行筛选，刷新数据。

![效果图](https://github.com/Veken/Expandablelistview/raw/master/screenshot/pic.gif) 

时间选择器还可以进行城市三级列表的选择，时间也可以扩充到日。
